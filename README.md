# 📄 SCED - Sistema de Controle de Entrada de Documentos

## 📖 Sobre o Projeto

O **SCED (Sistema de Controle de Entrada de Documentos)** é uma aplicação web desenvolvida para auxiliar no gerenciamento e controle de documentos e protocolos, permitindo o cadastro, consulta e acompanhamento das atividades realizadas pelos usuários de forma organizada e intuitiva.

O sistema foi desenvolvido como projeto acadêmico do curso de **Análise e Desenvolvimento de Sistemas (ADS)**, utilizando arquitetura cliente-servidor e banco de dados relacional para armazenamento das informações.

---

## 🚀 Funcionalidades

- 🔐 Autenticação de usuários;
- 📄 Cadastro e gerenciamento de documentos;
- 📂 Upload de arquivos;
- 📑 Controle de protocolos;
- 📊 Dashboard com indicadores do sistema;
- 📈 Relatórios com filtros por período, status e tipo de documento;
- 📋 Histórico de Logs e Auditoria;
- 🗄️ Integração com banco de dados MySQL.

## ▶️ Como executar o projeto

### Clonar o repositório

```bash
git clone https://github.com/RafaelaMichelin/ProjectFileManagment.git
```

### Acessar a pasta

```bash
cd ProjectFileManagment
```

### Instalar as dependências do Front-end

```bash
npm install
```

### Instalar as dependências do Back-end

```bash
npm install --prefix server
```

### Executar o projeto

```bash
npm run dev
```

O comando acima inicializa simultaneamente o Front-end e o Back-end da aplicação.

## 📊 Principais Módulos

### 🏠 Dashboard

Tela inicial responsável por apresentar indicadores gerais do sistema através de gráficos, tabelas e informações resumidas sobre documentos e protocolos cadastrados.

### 📈 Relatórios

Permite visualizar informações utilizando filtros por período, status e tipo de documento, exibindo gráficos e tabelas dinâmicas para facilitar a análise dos dados.

### 📋 Logs

Responsável pela auditoria das ações realizadas pelos usuários, permitindo consultar eventos registrados, filtrar informações e acompanhar o histórico de atividades do sistema.
